# 朋友圈人格分析 H5 网页需求文档 (PRD)

## 1. 项目概述
本项目旨在开发一个极简的 H5 单页面应用，用户通过上传 5 张或以上的朋友圈截图，利用豆包大模型（doubao-seed-1.6-thinking）进行深度社交分析。核心目标是帮助用户洞察对方心理，提供精准的追求建议，助力建立恋爱关系。

## 2. 核心目标
- **极简交互**：单页面完成上传与结果展示。
- **高效助攻**：基于大模型输出简短、犀利、有效的追求策略。
- **精准洞察**：利用 AI 深度解析朋友圈背后的心理状态。

## 3. 用户流程 (单页面)
1. **上传区域**：页面顶部提供明显的图片上传入口，支持 5-10 张截图。
2. **分析触发**：上传满 5 张后自动或手动触发“开始分析”。
3. **加载态**：在当前页面展示分析进度。
4. **结果展示**：分析结果直接渲染在上传区域下方。
   - **分析报告**：包括性格画像、追求技巧、禁忌提醒等。

## 4. 功能需求
### 4.1 图片上传
- **数量限制**：最少 5 张，上限 10 张。
- **格式支持**：JPG, PNG, WebP。
- **预览功能**：上传后可查看缩略图并支持删除/重传。

### 4.2 AI 分析 (核心)
- **模型调用**：使用 `doubao-seed-1.6-thinking` 模型。
- **System Prompt**：
  > 你是一个追求异性的高手，对于青春男女生的心理和外在表现，有非常强的洞察，也有一套很厉害的追求异性的技巧！擅长于输出简短但有效的分析和建议。
- **输入要求**：通过 OCR 识别（或模型直接读取图片）获取朋友圈内容，结合 Prompt 进行分析。

### 4.3 结果呈现
- 结果以简洁的卡片形式展示。
- 提供“重新分析”按钮。

## 5. 技术架构
- **前端**：HTML5, Tailwind CSS, JavaScript (React 或 Vue)。
- **后端**：Node.js (API 转发与 Key 保护)。
- **大模型 API**：火山引擎方舟平台 (Volcengine Ark)。
  - 模型：`doubao-seed-1.6-thinking`。
  - Base URL: `https://ark.cn-beijing.volces.com/api/v3`。

## 6. 非功能需求
- **单页体验**：无跳转，状态流转平滑。
- **响应式**：适配移动端主流浏览器。
- **隐私保护**：图片仅用于分析，不进行持久化存储。
